--- title: Cheat sheet de punto flotante para C# description: Trucos para usar punto flotante y números decimales en C# --- Tipos de punto flotante ----------------------- C# tiene tipos de precisión sencilla y doble del [IEEE 754](/formats/fp/) soportados por palabras clave: float f = 0.1f; // punto flotante de 32 bits, nótese el sufijo f double d = 0.1d; // punto flotante de 64 bits, sufijo opcional Tipos decimales --------------- C# tiene un tipo decimal de 128 bit de [precisión limitada](/formats/exact/) denotado con la palabra clave `decimal`: decimal myMoney = 300.1m; // Nótese el sufijo m en el literal Cómo redondear -------------- El método `Math.Round()` funciona con los tipos `double` y `decimal`, y permite especificar el [método de redondeo](/errors/rounding/): Math.Round(1.25m, 1, MidpointRounding.AwayFromZero); // Devuelve 1.3 Recursos -------- * [C# Reference](http://msdn.microsoft.com/en-us/library/618ayhy6%28v=VS.80%29.aspx) * [float type](http://msdn.microsoft.com/en-us/library/b1e65aza%28v=VS.80%29.aspx) * [double type](http://msdn.microsoft.com/en-us/library/678hzkk9%28v=VS.80%29.aspx) * [decimal type](http://msdn.microsoft.com/en-us/library/364x0z75%28v=VS.80%29.aspx) * [Math.Round()](http://msdn.microsoft.com/en-US/library/system.math.round%28v=VS.80%29.aspx)
